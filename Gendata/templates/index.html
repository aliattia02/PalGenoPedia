Help & Instructions

How to Use the Gaza Crisis Data Extractor

Adding URLs
  • Enter a single URL in the "Single URL" field and click "Add"
  • Or paste multiple URLs in the "Bulk URLs" textarea, one per line
  • Use the "Validate" button to check if a URL is accessible
  • Click the example URL to see the expected format
Data Extraction
  • Review your URL list before starting extraction
  • Choose whether to update the main CSV file
  • Enable backup creation for safety
  • Click "Start Extraction" to begin processing
File Management
  • View all extracted CSV files in the File Management section
  • Download files directly from the browser
  • Preview file contents before downloading
  • Check extraction statistics
Supported Sources
  • Al Jazeera (primary support)
  • BBC News
  • Other news sites (basic extraction)
